# Dice Poker (サイコロポーカー)

学校の授業で作成した、サイコロを振って役を作り高スコアを目指すストラテジーゲームです。
その後、**BGMや効果音の追加**、**キー入力の改善**、**GitHub Pagesでの公開**などの修正を行い、完成度を高めました。

## 📁 提出資料一覧
このリポジトリには、ゲームのソースコードに加えて、以下の設計・企画資料一式が同梱されています。
* **企画書 (`DicePoker_企画書.pptx`)**: ゲームの概要、特徴、サイクルの定義
* **設計書 (`DicePoker_設計書.xls`)**: 各画面の画面遷移図およびレイアウト設計
* **機能・データ設計書 (`DicePoker_機能データ設計.docx`)**: 各画面のロジック、役の判定アルゴリズム等の定義
* **操作マニュアル (`DicePoker_操作マニュアル.docx`)**: キー操作方法の解説

## 🎮 ゲーム概要
* **ルール**: 5個のサイコロを振り、ポーカーのルールに基づいた役を作ってスコアを獲得します。
* **特徴**: 任意のサイコロを選択して最大3回まで振り直せる「リロール機能」を搭載しており、プレイヤーの戦略性が問われます。

## 🛠 使用技術
* HTML5 / Canvas
* JavaScript (ES6)

## ⌨️ 操作方法
* `Space`: ゲームスタート / ロール / 役の確定
* `1` 〜 `5`: 1〜5個目のサイコロを Hold（キープ） / Change（振り直し）に切り替え
* `R`: Changeに設定したサイコロをリロール（最大3回）
* `Esc`: リトライ（タイトルへ戻る）
